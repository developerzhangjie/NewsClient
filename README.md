# 新闻客户端

##基本信息

- 开发环境：Android Studio2.2.3
- 运行环境：Android4.3 Android4.4.4 Android5.0
- 测试机型：中兴U9180 小米3 华为荣耀C8817D NEXUS5X
- 使用框架：SlidingMenu、ViewPagerIndicator、XUtils3、极光推送、社会化分享等
- 服务端：使用Tomcat9.0，JSON数据是从网络上找的
